# ГЛУБИНА 13 2.0.0

Экспедиционный 3D-хоррор для Pocket Works. Игрок управляет небольшой подводной капсулой, исследует бесконечные процедурные руины, находит сокровища сонаром и должен физически вернуться с ними к стартовому шлюзу.

## Основной цикл

1. Капсула выходит из шлюза с полным воздухом и пустыми грузовыми креплениями.
2. Левый аналоговый джойстик управляет движением: малое отклонение даёт тихий ход, полное — высокую скорость и заметный акустический след.
3. Сонарная кнопка справа создаёт объёмную волну, которая проявляет геометрию, добычу, чёрный ящик и Слушателя.
4. Манипулятор автоматически забирает находку, когда капсула подходит близко и сбрасывает скорость.
5. Экспедиция завершается только после возвращения в шлюз и двух секунд неподвижной фиксации.
6. Доставленная добыча превращается в фонд станции, который тратится на тихий винт, дополнительные секции корпуса и грузовые крепления.

## Хоррор-система

Слушатель не патрулирует игрока по прямому скрипту. Он просыпается из-за накопленного внимания и идёт к последнему значимому звуку:

- активному сонару;
- быстрому ходу;
- столкновению со стеной;
- работе манипулятора и повреждению корпуса.

Когда капсула замолкает, Слушатель продолжает проверять старую точку шума, а не получает магическое знание координат игрока. Его модель почти невидима без сонарной волны; приближение передаётся звуковыми откликами и изменением интерфейса.

## Добыча

- **Реликт** — частая находка, 30 единиц фонда.
- **Архив** — редкая техническая запись, 70 единиц.
- **Идол** — опасная и дорогая находка, 150 единиц.
- **Кислородный модуль** — восстанавливает воздух и часть заряда сонара.
- **Чёрный ящик** — сохраняет одну самую дорогую находку после гибели и возвращает следующую экспедицию в тот же процедурный сектор.

## Техническая структура

- Runtime: Quick PWA / canvas.
- Рендер: собственный WebGL 1 движок с перспективной камерой, примитивными мешами, туманом, подсветкой граней и объёмной сонарной волной.
- Процедурная генерация: детерминированные чанки 24×24 метра, загружаемые вокруг игрока.
- Управление: Pointer Events, клавиатурный fallback WASD/стрелки, пробел для сонара.
- Звук: процедурный Web Audio; вибрация отключается отдельно.
- Хранение: `pocket-works:glubina-13:*`.
- Офлайн: собственный Service Worker с префиксом `glubina-13-`.
