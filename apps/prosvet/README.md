# ПРОСВЕТ 1.0.0

Вертикальная кинетическая аркада для Pocket Works. Игрок вращает активное кольцо прямым жестом, совмещает его прорезь с фиксированной иглой и проводит импульс к центру до того, как погаснет пульс.

## Игровой цикл

1. Пальцем вращается только активное кольцо; быстрый бросок передаёт ему инерцию.
2. Кнопка «Пройти» проверяет реальное совпадение прорези с иглой.
3. Точное совпадение даёт больше очков и энергии, ошибка обнуляет серию и отнимает пульс.
4. После открытия ядра создаётся новый процедурный слой: кольца ускоряются, а прорези становятся уже.
5. Забег заканчивается при нулевом пульсе; лучший результат сохраняется локально.

## Интерфейс и доступность

- Полный текстовый возврат в Pocket Works есть на стартовом экране, в паузе и после проигрыша.
- Первый запуск содержит короткое встроенное обучение.
- Звук, вибрация и движение отключаются независимо.
- `prefers-reduced-motion` отключает инерционные переходы по умолчанию.
- Незавершённый забег сохраняется и может быть продолжен после перезапуска.

## Техническая структура

- Runtime: Quick PWA / `game-2d`.
- Рендер: адаптивный high-DPI Canvas 2D.
- Звук: процедурный Web Audio без внешних файлов.
- Хранение: уникальные ключи `pocket-works:prosvet:*`.
- Офлайн: собственный Service Worker и префикс кэша `prosvet-`.
- Workshop Mode подключён через общий модуль Pocket Works.
