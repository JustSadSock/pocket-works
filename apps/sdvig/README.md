# СДВИГ 1.0.0

Абстрактная соревновательная игра для Pocket Works. Игрок не ставит фишку в свободную клетку, а вталкивает её с одного из двадцати краевых портов. Выбранная строка или колонка сдвигается целиком, а крайняя фишка выпадает.

## Правила

- Поле: 5×5.
- Киноварь соединяет левый край с правым.
- Кобальт соединяет верхний край с нижним.
- За ход игрок вталкивает свою фишку с любой стороны в любую строку или колонку.
- Нельзя немедленно протолкнуть ту же линию обратно.
- После первого хода второй игрок может поменяться цветами по правилу пирога.
- Если твой сдвиг завершил цепь соперника, раунд получает соперник.

## Режимы

- локальная дуэль на одном устройстве;
- тренировка против «Станка» с глубиной поиска 1–3;
- матчи до одной, двух или трёх побед;
- без часов, 10 или 5 секунд на ход.

## Техническое

- чистые HTML, CSS и JavaScript;
- состояние поля и матч сохраняются в `pocket-works:sdvig:profile`;
- процедурный Web Audio без внешних файлов;
- вибрация с безопасным fallback;
- собственные manifest, service worker, cache namespace и SVG-иконка;
- мобильная портретная раскладка и компактная ландшафтная версия.
