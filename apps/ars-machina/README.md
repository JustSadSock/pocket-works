# ARS MACHINA

Мобильная 2D-песочница Pocket Works для сборки средневековых механизмов из универсальных физических деталей.

## Основной цикл

1. Выбрать деталь на нижней инструментальной рейке.
2. Поставить узлы и соединить их балками, тросами, пружинами или защёлками.
3. Добавить колёса, шестерни, моторы, лебёдки, противовесы и снаряды.
4. Перейти в режим испытания, управлять приводом и активировать спуск.
5. Вернуться в мастерскую, поправить конструкцию и повторить.

## Управление

- **Выбор** — перемещение узлов и настройка выбранной детали.
- Если перетащить один конец балки на другой узел, они автоматически сольются в общий подвижный шарнир.
- **Шарнир** — первое касание выбирает конец одной балки, второе касание объединяет его с другим близким концом; касание пустой сетки создаёт новый узел.
- **Ладонь** — перемещение камеры.
- Балки, тросы, пружины и защёлки используют два последовательных касания.
- `＋`, `−` и `⌂` управляют камерой.
- Состояние автоматически сохраняется в `pocket-works:ars-machina:blueprint`.

В комплект входят полностью редактируемые чертежи зубчатой телеги и малого требушета.
