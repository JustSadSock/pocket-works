# БЛАЗОН: Доктрина

Автономный геральдический автобаттлер для Pocket Works.

## Living Battle v4

- восемь отрядов на сторону остаются командными единицами;
- внутри них существуют 32 отдельных ратника и 16 отдельных лучников;
- каждый воин имеет собственные координаты, скорость, цель, состояние, здоровье и мораль;
- бойцы физически заполняют бреши, перестраивают ряд вокруг потерь и расходятся при локальном давлении;
- ближний бой состоит из индивидуальных контактов, а стрелы летят к конкретным целям;
- падение, попадание, отступление, движение и колебание знамени отображаются отдельно;
- геральдическая доктрина продолжает управлять отрядами, но её результат теперь исполняется живыми воинами.
