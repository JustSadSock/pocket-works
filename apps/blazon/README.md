# БЛАЗОН: Доктрина

Автономный геральдический автобаттлер для Pocket Works. Версия 3.1 добавляет отдельный визуальный слой Heraldic Identity без изменения боевой механики.

## Heraldic Identity

- четыре композиционные школы, связанные с ординариями;
- четыре разных силуэта щита;
- поле, металл, главная фигура и отдельный акцент вместо двухцветного слияния;
- разные схемы размещения главной и вторичной фигур;
- внешние украшения школы и командного слоя;
- перенос палитры и знаков герба на знамёна, щиты ратников и экипировку лучников;
- предпросмотр награды как цельного будущего герба.
