# БЛАЗОН: Доктрина

Автономный геральдический автобаттлер для Pocket Works.

## Formation Physics 4.2

- поле развёрнуто сверху вниз под вертикальный мобильный экран;
- восемь отрядов на сторону остаются командными единицами;
- 96 отдельных воинов образуют мягкие построения и получают свободные места внутри своего отряда;
- после потерь ближайшие бойцы заполняют бреши без телепортации и пересечения рядов;
- союзники и противники физически раздвигаются, поэтому линия не превращается в одну наложенную кучу;
- фронт замедляет продвижение связанного отряда, а отдельные пары ведут локальный ближний бой;
- камера мягко следует за первым контактом и решающим участком;
- HUD и события не закрывают центральную схватку.
