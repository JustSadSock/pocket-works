# БЛАЗОН: Доктрина

Автономный геральдический автобаттлер для Pocket Works.

## Ядро

- Равные армии: 32 ратника, 16 лучников и знамя на сторону.
- Игрок не управляет боем после его начала.
- Шесть слоёв герба задают темперамент, построение, поведение ратников, поведение лучников, командование и одно условное исключение.
- Элементы меняют решения и движение, но не урон, здоровье, скорость или дальность.
- Единственный формат: полевая битва до захвата знамени или распада трёх из четырёх отрядов.
