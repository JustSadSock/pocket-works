# БЛАЗОН: Доктрина

Автономный геральдический автобаттлер для Pocket Works.

## Armorial Progression v5

- первый герб начинается не пустым: игрок до первого боя выбирает поле, ординарий, фигуру-основателя и независимую визуальную школу;
- школа больше не связана с механикой строя и отвечает только за форму щита, композицию и внешнее достижение;
- внешние элементы раскрываются по прогрессии: простой щит → вторичные знаки → командование → девиз → полное достижение;
- после четвёртого боя основатель получает одну из трёх эволюций, которая меняет механику и собственный геральдический силуэт;
- всего реализовано 12 эволюций для Льва, Вепря, Башни и Оленя;
- звериные фигуры перерисованы вручную в едином более читаемом геральдическом языке;
- существующие системы Living Battle, Formation Physics и Battle Reading сохранены.
