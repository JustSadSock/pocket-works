# БЛАЗОН: Доктрина

Автономный геральдический автобаттлер для Pocket Works.

## Battle Reading 4.3

- лучники готовят и выпускают согласованные залпы отделениями, а не стреляют независимо по личному таймеру;
- Солнце концентрирует огонь, Орёл ведёт залпы с разведённых позиций, Роза стреляет из-под прикрытия, Ключ ждёт открытого пролома;
- поле, фигуры, командование и девизы показывают краткие сигналы непосредственно возле затронутых воинов;
- бой записывается компактными снимками без попадания тяжёлого повтора в постоянное сохранение;
- результат выбирает три ключевых момента и воспроизводит их в отдельной миниатюрной сцене;
- причинный разбор строится из фактических распадов, попаданий, срабатываний и способа завершения битвы;
- мобильная вёрстка результата больше не обрезает итог и складывает действия вертикально.
