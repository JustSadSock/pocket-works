# НУЛЕВОЙ КАДР

Иммерсивная мобильная хоррор-игра, где телефон работает как проклятый моментальный фотоаппарат.

## Игровой цикл

1. Осматривать панорамную комнату наклоном телефона, перетаскиванием пальца или стрелками/WASD.
2. Искать аномалию по криптической подсказке и снимать её кнопкой затвора или пробелом.
3. Перевернуть телефон экраном вниз, чтобы проявить фотографию. Если датчик движения недоступен, удерживать экранную кнопку.
4. Проверить проявленный кадр. Верные снимки открывают следующую аномалию; ошибки расходуют плёнку и повышают страх.
5. Снять пять меток, включая появившуюся дверь, до окончания семи кадров или заполнения шкалы страха.

## Особенности

- Canvas-панорама с процедурной комнатой, перспективой, фонарём, зерном, пылью и угрозой в периферии.
- Device Orientation для обзора и Device Motion для определения переворота экраном вниз.
- Полноценный сенсорный, мышиный и клавиатурный резерв без урезания механики.
- Процедурный Web Audio без внешних файлов, отключаемый звук и прогрессивная вибрация.
- Локальное сохранение активной плёнки, статистики побегов и лучшего времени.
- Самостоятельный PWA-кэш, управляемые обновления и Workshop Mode.

## Хранилище

- Namespace: `pocket-works:zero-frame`
- Основной ключ: `pocket-works:zero-frame:state`

## Ограничения платформы

На iOS доступ к наклону и движению запрашивается только после нажатия кнопки запуска. В браузерах без этих API игра автоматически использует пальцевое управление и экранную кнопку проявления.
