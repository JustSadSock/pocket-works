# МЕССА

«МЕССА» — автономная мобильная 3D-игра о небесной машине. Комета живёт по собственной орбитальной физике; игрок не рулит ею напрямую, а удержанием пальца создаёт временную сингулярность в выбранной точке пространства.

## Игровой цикл

1. Удерживать палец в мире и двигать гравитационный колодец, искривляя траекторию кометы.
2. Собирать восемь голосов по порядку и пролетать близко к движущимся монолитам ради энергии и очков.
3. Тратить накопленную энергию на ручную перемотку опасного участка.
4. Потеря устойчивости запускает аварийный откат; после трёх ударов попытка заканчивается.
5. Сбор восьмого голоса пробуждает машину и завершает миссию.

## Управление

- Палец / мышь по сцене — создать и переместить сингулярность.
- «Откат» или `R` — ручная перемотка, когда накоплено достаточно энергии.
- `Space` — удерживать сингулярность в позиции курсора.
- `Esc` — пауза.

## Технически

- Собственный WebGL-рендер без сетевых ассетов и стороннего runtime.
- Детерминированная орбитальная физика и движущиеся препятствия.
- Процедурный Web Audio, локальные рекорды, адаптивное разрешение и PWA-кэш.
- Приложение полностью изолировано внутри `apps/messa/`.
