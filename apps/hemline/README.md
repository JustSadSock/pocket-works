# ШОВ 1.0.0

«ШОВ» — офлайн-абстрактная стратегия для Pocket Works. Игрок охры соединяет верхний и нижний края поля 8×8, игрок индиго — левый и правый.

## Правила

- За ход ставится один камень.
- Ровно два чужих камня, зажатые по прямой между двумя своими, снимаются. Один или три камня не снимаются.
- У каждого игрока два сдвига: вместо постановки можно передвинуть свой камень на соседнюю пустую клетку.
- После первого хода второй игрок может перехватить открывающий камень, перекрасив его в свой цвет и вернув ход сопернику.
- Повтор уже встречавшейся позиции запрещён.
- Побеждает непрерывная цепь между назначенными краями.

## Режимы

- локальная дуэль на одном устройстве;
- офлайн-бот с четырьмя характерами: Штурм, Заслон, Охотник и Ткач;
- сохранение незавершённой партии после каждого хода.

## Баланс

Перед релизом проверены 20 конфигураций размера поля, захватов, сдвигов и правила перехвата. Два финалиста получили отдельный турнир на 64 партии. Подробности — в `BALANCE_AUDIT.md` и `tests/`.

## Техническое

Quick PWA, Canvas 2D, Web Audio, локальное хранилище Pocket Works, Workshop Mode и собственный Service Worker `hemline-v1.0.0-p2`.
