Как играть

Четыре правила

  1. Соедини края.Охра строит путь сверху вниз. Индиго — слева направо. Соседство по шести сторонам гекса.
  2. Зажми ровно пару.Два чужих камня между твоими по прямой снимаются. Один или три не снимаются.
  3. Два сдвига.Вместо постановки передвинь свой камень в соседнюю пустую клетку. Сдвиг тоже может захватить пару.
  4. Перехват открытия.После первого хода второй игрок может перекрасить этот камень в свой цвет и вернуть ход сопернику.
