# ПОЛЕВОЙ ШТАБ 1.0.0

Мобильная процедурная стратегия для Pocket Works. Игрок командует армией на тканевой штабной карте: выбирает отряды, отдаёт приказы, использует рельеф, снабжение, мораль, укрепления и разведку, захватывает штандарты и продолжает кампанию без фиксированного финала.

## Основной цикл

1. Перед кампанией выбирается одна из трёх доктрин: манёвр, огневой вал или несгибаемость.
2. В каждом бою игрок распоряжается ограниченным запасом очков приказов и пятью типами войск.
3. Победа достигается через восемь очков контроля, уничтожение армии или поломку вражеского строя.
4. После победы выбирается одно из трёх постоянных улучшений штаба.
5. Следующий процедурный бой усиливает противника и меняет рельеф, погоду и расклад сил.

## Продуктовые детали

- Визуальный язык полевой карты: ткань, деревянные жетоны, карандашные контуры, штабные папки и печати.
- Полноценный вражеский ИИ: атака уязвимых целей, движение к штандартам, укрепление ослабленных частей.
- Туман разведки скрывает отдалённые вражеские отряды.
- Активная кампания, настройки и рекорд сохраняются локально.
- Процедурные Web Audio-звуки, отключаемая вибрация и reduced motion.
- Отдельная PWA-иконка, автономный Service Worker, Workshop Mode и явный возврат в Pocket Works.
