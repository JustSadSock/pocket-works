# СВИСТОК 90

Вертикальный мобильный футбольный симулятор 7×7 для Pocket Works. Матч длится около шести минут и включает два тайма, добавленное время, полноценные рестарты, офсайды, фолы, преимущество, карточки, удаления, пенальти, VAR и замены.

## Управление

- левый джойстик — движение выбранного игрока;
- **ПАС** — пас с мячом, подкат без мяча;
- **УДАР** — удерживание заряжает силу удара;
- **РЫВОК** — ускорение с расходом выносливости;
- **↻ ИГРОК** — смена управляемого футболиста.

## Визуальный принцип

Поле и интерфейс стилизованы под печатную спортивную графику: плоские краски, жёсткие рамки, бумажная фактура и телевизионные титры. VAR временно превращает матч в технический чертёж с линией офсайда.

## Технически

- Enhanced PWA: Vite, TypeScript, Phaser 3, Workbox и Vitest;
- вся графика процедурная, без внешних изображений;
- свистки, удары, трибуны и сигналы синтезируются через Web Audio;
- прогресс и настройки хранятся в `pocket-works:whistle-90:state`;
- приложение работает офлайн после первой успешной загрузки.
