# ЧЕРТА

Небольшая мобильная боёвка для Pocket Works, построенная вокруг одного жеста.

## Игровой цикл

1. Зажать палец на арене — время замедляется.
2. Протянуть палец и отпустить — герой делает рывок.
3. Каждый рывок оставляет вооружённую линию.
4. Третья линия запускает последовательный разрез арены.
5. Игрок переживает бесконечные круги, рискует ради чистых уклонений и повышает рекорд.

## Управление

- **Зажать + протянуть + отпустить** — направленный рывок.
- **Пауза** — кнопка `Ⅱ`.
- **Звук** — кнопка `♪` или переключатель в настройках.
- **Возврат в Pocket Works** — кнопка `⌂`.
- На клавиатуре: `Space` выполняет рывок вверх, `Escape` переключает паузу.

## Состояние и хранение

Локально сохраняются настройки звука и вибрации, лучший счёт и лучший достигнутый круг. Пространство ключей: `pocket-works:cherta:*`.

## Offline / PWA

Приложение использует собственный `manifest.webmanifest`, Service Worker и кэш `cherta-v1.0.0`. После первого успешного запуска игровой shell доступен без сети.

## Ограничения

Вибрация зависит от поддержки браузера; на iOS она может быть недоступна. Звук запускается только после осознанного действия пользователя.
