# ФАЗА

Офлайн-абстрактная дуэль для Pocket Works. Игрок ставит один камень и выбирает одну из трёх фаз связности. Фаза выключает одну ось соседства, после чего группы без свобод снимаются.

## Победа

- снять 5 камней соперника; или
- соединить свои противоположные края и сохранить путь после ответного хода.

Синий соединяет левый и правый край. Охра соединяет верхний правый и нижний левый. Самозахват запрещён. После первого хода второй игрок может обменяться сторонами.

## Режимы

- локальная дуэль на одном экране;
- одиночная игра против четырёх характеров ИИ;
- три уровня поиска;
- серии до 1, 2 или 3 побед с чередованием первого хода.

## Проверка

```bash
node --check apps/faza/game.js
node --check apps/faza/ai.js
node --check apps/faza/app.js
node apps/faza/tests/audit.mjs
```

Отчёт о 40 конфигурациях и более 2 600 бот-партий находится в `DESIGN-AUDIT.md` и встроен в экран «Баланс и тесты».

## Хранение и офлайн

- namespace: `pocket-works:faza`;
- cache: `faza-v1.0.0`;
- состояние серии, незавершённая партия и настройки сохраняются локально;
- все игровые и общие runtime-файлы входят в Service Worker shell.
