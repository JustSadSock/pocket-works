# ХРЕБЕТ

Процедурная мобильная 3D-аркада о полёте в вингсьюте по бесконечному горному разлому.

## Игровой цикл

1. Выбрать случайный маршрут или общий маршрут дня.
2. Вести пилота пальцем, мышью, стрелками или WASD.
3. Удерживать кнопку «Сложить» / Space, чтобы прижать крылья, нырнуть и набрать скорость.
4. Проходить сигнальные ворота и лететь близко к снегу, стенам и скальным иглам ради серии и потока.
5. При полной шкале потока пережить короткий ускоренный прорыв с двойными очками.
6. После разрушения крыла сравнить дальность, счёт, ворота, пролёты на грани и максимальную серию.

## Техническая схема

- Quick PWA без сетевых runtime-зависимостей.
- Собственный WebGL 1 рендерер с плоским освещением, туманом, прозрачными материалами и динамическими частицами.
- Детерминированная генерация рельефа, ворот и препятствий по seed; маршрут дня повторяется по локальной календарной дате.
- Процедурный Web Audio: ветер зависит от скорости и положения крыльев, события синтезируются на месте.
- Состояние активного полёта, настройки и рекорды хранятся только в `pocket-works:khrebet`.
- Service Worker кэширует весь игровой и общий Pocket Works runtime для автономного повторного запуска.
