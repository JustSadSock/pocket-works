# СГИБ

Мобильная головоломка для Pocket Works, где игрок не двигает фишки по полю, а физически складывает сам лист.

## Основной жест

Нажми на бумагу и протяни палец к точке назначения. Середина движения становится линией сгиба, направление движения определяет, какая половина листа поднимается и куда она ложится. Во время жеста показывается объёмный предварительный сгиб; после отпускания он завершается и меняет форму всех бумажных слоёв.

## Правила

- тёмная капля должна совпасть с пустым тиснёным кругом;
- две капли также взаимно гасят друг друга;
- совпавшие пары исчезают;
- лист нужно очистить за указанное число сгибов;
- слишком короткий жест или сгиб, не пересекающий бумагу, не расходует ход.

## Режимы

- **Кампания** — десять авторских листов, обучающих вертикальным, горизонтальным, диагональным и последовательным сгибам;
- **Бесконечная стопка** — процедурные листы, каждый из которых гарантированно решается одним точным сгибом. Лучший результат сохраняется локально.

## Техническая часть

Приложение использует Canvas 2D, Pointer Events, shared mobile runtime, versioned local storage, Workshop Mode, процедурный Web Audio и собственный Service Worker. Оно не требует сети после первой успешной загрузки.
