# РАТЬ

Мобильная тактическая дуэль для Pocket Works. Игрок расставляет три полка, после чего бой идёт на уровне отдельных маленьких бойцов.

## Основной цикл

1. Переставить мечников, копейщиков и лучников между левым флангом, центром и правым флангом.
2. Выбрать для каждого полка линию, клин или рассыпной строй.
3. Начать бой против случайно сформированной армии ИИ.
4. В любой момент переключаться между автономным наблюдением, редкими приказами и свободными полковыми флагами.
5. Победить, уничтожив или обратив в бегство армию противника.

## Архитектура управления

Симуляция бойцов не зависит от интерфейса командования. Режимы управления меняют только источники полковых целей, поэтому новые способы игры можно добавлять без переписывания боя.

## Хранение

Настройки и последняя расстановка сохраняются в `pocket-works:rat:state`.
