# ЛИПУЧКА

Однопальцевая антистресс-игра для Pocket Works. Игрок водит липким валиком по процедурной ткани, собирает ворс, снимает заполненный слой и переходит к следующему полотну.

## Основной цикл

1. Вести пальцем по экрану — валик следует немного выше касания.
2. Собирать ворс и удерживать серию движением без длинных пауз.
3. Когда валик заполнен, нажать или потянуть вверх кнопку снятия слоя.
4. Получить ворс, завершить очистку ткани и открыть следующий узор.
5. Тратить ворс в мастерской на ширину, липкость и автопилот.

После нескольких секунд бездействия включается медленный автопилот. Поражений, таймеров, рекламы и внешних ресурсов нет.

## Техническое

- Quick PWA: HTML, CSS, Canvas 2D и JavaScript без зависимостей.
- Локальное сохранение: `pocket-works:lipuchka:v1`.
- Уникальный offline-кэш: `lipuchka-v1.0.0`.
- Процедурные ткань, ворс, частицы, звук и вибрация.
- Игровой цикл останавливается, когда страница скрыта.
