# ОТТИСК

Однопальцевая мобильная игра для коротких фоновых сессий. Игрок ловит три движущихся ризографических слоя в регистрационной метке, после чего получает физически несовершенный многокрасочный оттиск.

## Основной цикл

1. Первый красочный слой движется вокруг центра.
2. Тап фиксирует слой и оценивает точность совмещения.
3. После трёх слоёв печатается один оттиск.
4. Восемь оттисков завершают тираж и сохраняют рекорд.
5. Каждые два завершённых тиража открывается дополнительная палитра.

Поражения нет: неточный тап создаёт более выраженный печатный брак, но игра продолжается. Высокая точность поддерживает серию и увеличивает итоговый счёт.

## Управление

- Тап по игровому полю — зафиксировать текущий слой.
- `ЗВУК / ТИХО` — включить или отключить синтезированный звук.
- `ЕЩЁ ТИРАЖ` — начать новый цикл после финала.
- `Сбросить прогресс` — удалить рекорд и открытые палитры после подтверждения.
- `PW` — вернуться в оболочку Pocket Works.

## Хранение и офлайн

Состояние хранится в `localStorage` под ключом `pocket-works:ottisk:state:v1`. Service Worker использует изолированный кэш `ottisk-v1.0.0` и сохраняет только файлы приложения и явно подключённые модули Pocket Works.
