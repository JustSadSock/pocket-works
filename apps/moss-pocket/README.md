# МОХ

Живая карманная экосистема для коротких спокойных сессий. Игрок ведёт влагу пальцем по колонии, собирает росу, вызывает дождь и после каждого цикла выбирает постоянную мутацию.

## Игровой цикл

- касания и медленные проводки дают росу и рост;
- дождь обменивает 25 росы на 22% роста;
- созревшая колония предлагает три реальные мутации;
- пассивный и офлайн-рост продолжают цикл между запусками;
- состояние, звук и прогресс сохраняются локально.
