# ОРБИТА

Локальная абстрактная PvP-игра для двух игроков на одном устройстве.

## Правила

Поле состоит из четырёх концентрических колец по восемь ячеек. За ход игрок:

1. ставит камень своего цвета в любую свободную ячейку;
2. обязательно поворачивает любое кольцо на один сектор по или против часовой стрелки.

Побеждает тот, чья непрерывная цепь соединит внутреннее кольцо с внешним. Соседними считаются ячейки рядом на одном кольце и ячейки в одном секторе на соседних кольцах. Победа проверяется после обязательного вращения.

После первого полного хода второй игрок может применить правило пирога и забрать цвет первого игрока. Матч идёт до трёх побед, начинающий игрок чередуется между раундами.

## Управление

- Нажмите свободную ячейку, чтобы поставить камень.
- Выберите кольцо I–IV и используйте кнопки вращения.
- Вместо кнопок можно провести пальцем вдоль нужного кольца.
- Настройки позволяют отключить звук и вибрацию, перезапустить раунд или сбросить матч.

## Техническое устройство

- Quick runtime: HTML, CSS и ES-модули без внешних зависимостей.
- Игровые правила вынесены в `engine.js` и покрыты тестами `node --test tests/engine.test.mjs`.
- Текущий матч и настройки сохраняются в локальном хранилище `pocket-works:orbita:*`.
- Service Worker кэширует только файлы приложения и обеспечивает офлайн-запуск.
