# КОНТУР

Бесконечная мобильная игра для Pocket Works о нестабильной электрической схеме.

## Игровой цикл

- Каждый каскад создаёт новую плату 6×8 с гарантированным маршрутом от нижнего контакта к верхнему.
- Импульс движется автоматически. Игрок касается модулей, поворачивая их на 90°.
- Разрыв, выход за край или петля сжигают предохранитель и возвращают импульс к источнику.
- После каждого четвёртого каскада игрок выбирает один из двух аппаратных модулей. Модули меняют скорость, запас ошибок, подсказки и множитель результата до конца забега.
- Скорость и сложность растут без конечного уровня.

## Управление

- Тап по модулю — поворот по часовой стрелке.
- Кнопка `Ⅱ` — пауза.
- Клавиатура: стрелки выбирают модуль, `Space`/`Enter` вращает, `Escape` ставит игру на паузу.

## Хранение

Приложение использует пространство `pocket-works:kontur`. Сохраняются рекорды, настройка звука, статистика и текущий забег. После перезагрузки незавершённый забег предлагается продолжить в безопасной паузе.

## Визуальная идея

Испытательный стенд из керамических модулей, окисленной зелёной платы, медных дорожек и механических счётчиков. Интерфейс намеренно избегает стекломорфизма, неона, плавающих карточек и универсальных капсул.

## Публикация

Версия 1.0.0 зарегистрирована в основной полке Pocket Works и должна публиковаться вместе с актуальной сборкой ветки `main`.
