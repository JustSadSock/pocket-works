- [x] Has a distinct launcher icon and portrait manifest.
- [x] Returns to the Pocket Works launcher through a persistent shell control.
- [x] Uses shared mobile runtime, versioned storage, managed updates and Workshop Mode.
- [x] Caches its complete application shell and game payload for offline use.
- [x] Handles fatal initialisation with a readable recovery screen.
- [x] Provides loading, empty, warning, modal, event and game-over states.
- [x] Simulation content and seeded generation pass local smoke validation.
- [x] Browser E2E covers quarter progression, persistence and Workshop diagnostics.
- [x] Payload integrity test verifies the committed compressed simulation bundle.
