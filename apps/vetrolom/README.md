# ВЕТРОЛОМ 1.3.0

Пиксельная survival RPG-песочница для Pocket Works.

## Основной цикл

Исследовать процедурную долину, переживать ночь и непогоду, собирать ресурсы, создавать инструменты, строить лагерь, выращивать пищу и ремонтировать три погодные вышки. После восстановления сети открывается Сердце Бури — финальная цель, после которой мир остаётся доступен как песочница.

## Что добавляет версия 1.3

- Новые естественные находки: грибы, валежник, полевые цветы, смола и редкие чистые источники.
- Умные действия лагеря: костёр жарит грибы, когда персонаж голоден и есть сырьё; днём лежанка даёт короткий отдых, ночью — полноценный сон.
- Штормовой фонарь создаётся из дерева, лома, смолы и кристалла. Его можно включать и выключать; активный свет отпугивает грозовых духов.
- Грибы на углях стали отдельной едой, которую можно приготовить рядом с костром.
- Подбор ресурсов отображается компактной лентой и частицами, летящими к персонажу, без текста поверх места действия.
- В мире появились следы, светлячки, птицы, дым, искры, водяные круги, движение травы и более выразительные тени времени суток.
- Фоновый звук реагирует на биом, дождь, бурю, ночь и состояние персонажа.

## Управление

- телефон: виртуальный стик, контекстное «ДЕЙСТВИЕ», «РЫВОК», панель инструментов;
- удержание «ДЕЙСТВИЯ» в режиме ограды прокладывает непрерывную цепь по мере движения;
- клавиатура: `WASD` / стрелки, `E` действие, `Space` рывок, `Q` смена инструмента, `I` рюкзак, `C` ремесло, `J` журнал, `Esc` пауза.
