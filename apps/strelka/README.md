# СТРЕЛКА

Карманная однопальцевая аркада про железнодорожный диспетчерский пульт.

## Игровой цикл

1. Состав появляется сверху и показывает платформу назначения: А, Б или В.
2. Игрок перетаскивает массивный рычаг в соответствующее положение до линии блокировки.
3. Верный маршрут увеличивает счёт и серию. Ошибка сжигает одну из трёх сигнальных ламп.
4. Темп постепенно ускоряется; экспрессы идут быстрее, грузовые составы дают больше очков.
5. После трёх ошибок смена заканчивается, рекорд сохраняется локально.

## Управление

- Перетаскивание рычага влево, по центру или вправо.
- Нажатие на буквы А / Б / В как доступная альтернатива.
- Пауза, звук и вибрация доступны из интерфейса.

## Технические детали

- Quick PWA без внешних зависимостей.
- Canvas с адаптивной плотностью пикселей.
- Процедурный Web Audio после пользовательского жеста.
- Локальные рекорды через `createVersionedStore`.
- Полная офлайн-оболочка и изолированный Service Worker `strelka-v1.0.0`.
