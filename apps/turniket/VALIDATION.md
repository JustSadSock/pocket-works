# Проверка версии 1.1.0

Выполнено перед публикацией:

```bash
node --check apps/turniket/app-v2.js
node --check apps/turniket/ai.js
node apps/turniket/ai.test.mjs
node apps/turniket/render.test.mjs
```

Дополнительно прогнаны пять смешанных партий до 60 полуходов: человек выбирался детерминированным генератором, ИИ использовал уровень «Ученик». Недопустимых действий и аварий движка не обнаружено.

Ориентировочное время первого решения на тестовой машине:

- Ученик: около 160 мс;
- Стратег: около 100 мс;
- Оракул: около 650 мс.

Физический прогон на iPhone из среды разработки недоступен. Регрессия чёрного SVG устранена архитектурно: все критические `fill` и `stroke` записываются прямо в создаваемые SVG-элементы буквальными значениями, без `var(...)` и без зависимости от внешних CSS-селекторов.
