# ТУРНИКЕТ

Локальная абстрактная PvP-игра для двух игроков на одном устройстве. По структуре хода она близка к Quoridor, но вместо глухих стен игроки ставят двухсекционные односторонние ворота. После каждого прохода весь турникет переворачивается.

## Правила

- Поле: 7×7 клеток.
- Игрок 1 начинает снизу и идёт к верхнему краю; Игрок 2 начинает сверху и идёт к нижнему.
- За ход игрок либо передвигает фишку, либо ставит один из семи турникетов.
- Турникет занимает две соседние щели, пропускает только в направлении стрелок и переворачивается после прохода через любую из двух секций.
- Нельзя поставить турникет, если текущая ориентация полностью уничтожает направленный маршрут хотя бы одного игрока к его цели.
- Фишку соперника можно перепрыгнуть. Если прямой прыжок закрыт краем или турникетом, разрешён диагональный обход.
- Раунд выигрывает игрок, который достиг противоположного края или оставил соперника без допустимого хода.
- Матч идёт до трёх побед; право первого хода чередуется по раундам.

## Управление

- **ХОД:** нажмите одну из подсвеченных клеток.
- **ВОРОТА:** нажмите тонкую метку между четырьмя клетками, выберите направление и подтвердите установку.
- Настройки позволяют включать и выключать синтезированный звук и виброотклик, переиграть раунд или сбросить матч.

## Техническое устройство

- Quick runtime: чистые HTML, CSS и JavaScript без внешних зависимостей.
- `game.js` содержит детерминированный движок, проверку направленной достижимости, столкновения ворот и правила прыжка.
- `app.js` отвечает за SVG-поле, ввод, сохранение и звуковую обратную связь.
- Состояние матча хранится в `pocket-works:turniket:match`.
- Service Worker кэширует только файлы приложения в пространстве `turniket-*`.
