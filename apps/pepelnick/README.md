# ПЕПЕЛЬНИК 1.0.0

Вертикальная пиксельная RPG-рогалик для Pocket Works.

- 7 залов, 4 типа врагов и двухфазный Хранитель Пепла;
- ручная серия ударов, рывок с i-frame, идеальное уклонение, Воля и казни;
- 18 реликвий и три постоянных улучшения;
- локальные сохранения, офлайн-PWA, процедурный звук и вибрация;
- управление с телефона и клавиатуры.

Управление: левый виртуальный стик, `УДАР`, `РЫВОК`; клавиатура — WASD/стрелки, J/Enter, K/Space, Escape.
