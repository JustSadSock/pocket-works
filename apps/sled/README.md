# СЛЕД 2.0.1

Абстрактная территориальная дуэль для Pocket Works.

## Правила

- Поле — правильный шестиугольник из 37, 61 или 91 соты.
- У каждого игрока собственный камень.
- Камень ходит на одну из шести соседних свободных сот.
- Покинутая сота приносит очко и становится стеной для обоих.
- Два автоматических паса подряд завершают партию.
- Второй игрок получает 0,5 очка коми.

## Исправление 2.0.1

Движок всегда использовал pointy-top axial-координаты: горизонтальные направления `E/W` и четыре наклонных направления. Однако CSS рисовал flat-top шестиугольники. Центры клеток стояли по одной геометрии, а контуры — по другой, поэтому поле соединялось углами и выглядело поломанным.

Теперь все игровые и декоративные соты используют один pointy-top контур. Центры соседних клеток и их общие грани совпадают во всех шести направлениях.
