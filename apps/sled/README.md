# СЛЕД 1.1.0

Абстрактная соревновательная игра для Pocket Works. На поле находится один общий камень, а каждая клетка имеет два состояния прочности: после первого ухода она трескается, после второго проваливается.

## Почему правила изменены

В версии 1.0 покинутая клетка исчезала мгновенно. Это создавало тупой инвариант: после первого вертикального шага соперник уже не мог вернуть камень назад, поэтому сторона направления открытия получала гарантированную победу. С включённым обменом этой стороной всегда становился второй игрок.

Версия 1.1 заменяет мгновенное сгорание на двойной излом. Теперь ответный игрок может развернуть движение через треснувшую клетку, но повторный уход уничтожит её.

## Правила

- Бирюзовая сторона пытается вывести камень через верхний край, коралловая — через нижний.
- За ход камень перемещается на одну соседнюю клетку по вертикали или горизонтали.
- После первого ухода клетка трескается, но остаётся проходимой.
- После второго ухода клетка проваливается и больше недоступна.
- Победа наступает после выхода через свой край или когда соперник остаётся без допустимого хода.
- Правило обмена отключено: двойной излом уже создаёт полноценный ответ на первый ход.

## Режимы

- локальная дуэль на одном устройстве;
- игра против ИИ: «Резчик», «Тактик», «Архитектор»;
- поля 7×7, 9×9 и 11×11;
- часы: без лимита, 3+2 и 10 минут.

## Управление

- касание соседней клетки;
- свайп по полю в нужную сторону;
- клавиатура: стрелки или WASD, Escape — пауза;
- выход через край выполняется нажатием на активную цветную рейку или свайпом наружу.

## Технические детали

- Quick PWA без внешних зависимостей;
- игровой движок в `engine.mjs`, состояния поля хранятся в двух BigInt-масках;
- миграция старых сохранений без сброса статистики и пользовательских настроек;
- локальные сохранения в `pocket-works:sled`;
- процедурный Web Audio, опциональная вибрация;
- собственный Service Worker, офлайн-загрузка и Workshop mode.
