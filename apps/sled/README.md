# СЛЕД 3.0.0

Абстрактная соревновательная игра о разрезании связности шестиугольного поля.

## Правила

- У каждого игрока один камень.
- За ход камень переходит на соседнюю свободную соту.
- Покинутая сота становится территорией игрока и непроходимой стеной.
- Любая свободная область, в которой не осталось камней, мгновенно обрушивается и целиком достаётся игроку, сделавшему разрез.
- Если стены разделили камни по разным связным областям, партия заканчивается сразу: остаток каждой области присваивается находящемуся в ней игроку.
- Если камню некуда двигаться, его владелец проигрывает ловушкой независимо от счёта.
- Второй игрок получает 0,5 коми для разрешения равного территориального раздела.

## Почему 3.0 не скучная зачистка

Один ход может забрать сразу несколько сот, создать угрозу ловушки или завершить партию. После решающего раздела не нужно доедать оставшееся поле: результат рассчитывается немедленно. Интерфейс показывает размер непосредственного захвата на легальных ходах, а решающие разрезы отмечает отдельно.

## Проверка

```bash
node --check app.js
node --check engine.mjs
node --check sw.js
node engine.test.mjs
node balance-audit.mjs
node smoke.test.mjs
```

Зафиксированный аудит из 1800 партий смешанных детерминированных ботов:

- 37 сот: первый 51,17%, средняя партия 19 ходов;
- 61 сота: первый 51,00%, средняя партия 30 ходов;
- 91 сота: первый 47,33%, средняя партия 43 хода.

В среднем происходят от 2,4 до 5 обрушений за партию, включая от 1,1 до 2,2 крупных захватов минимум по четыре соты.
