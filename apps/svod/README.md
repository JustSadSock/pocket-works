# СВОД

Мобильная стратегия о последнем городе, собранном из трёх вращающихся колец.

## Игровой замысел

Игрок напрямую вращает кольца города и совмещает районы в радиальные цепи. Правильные сочетания производят пищу, энергию, материал и волю; разорванные цепи оставляют внешний пояс без защиты перед стеклянной бурей. Между циклами игрок развивает районы, запускает городские проекты и принимает указы с отложенными последствиями.

Кампания процедурная и рассчитана на короткие, плотные партии с разными стартовыми хартиями, картой, угрозами и событиями.

## Идентичность

- Slug: `svod`
- Version: `0.1.0`
- Cache: `svod-v0.1.0`
- Storage: `pocket-works:svod`
- Orientation: `portrait`

## Границы приложения

Все исходники, визуал, звук, сохранения, PWA-манифест и offline shell принадлежат `apps/svod/`. Приложение использует общий мобильный runtime Pocket Works, но не меняет платформенные файлы.
