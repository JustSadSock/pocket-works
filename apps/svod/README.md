# СВОД

Мобильная стратегия о последнем городе, собранном из трёх вращающихся колец.

## Игровой цикл

Игрок напрямую вращает три кольца города и совмещает районы в радиальные цепи:

- внутренний привод производит воду, ток или труд;
- пояс мастерских превращает поток в паёк, детали или мандат;
- внешний город завершает цепь Кварталом, Бастионом или Советом.

Каждый цикл даёт ограниченный запас команд на повороты и одно городское дело: великий проект либо разовый приказ. После производства заранее отмеченные лучи атакует стеклянная буря. Работающий Бастион закрывает свой луч; повреждённые и разбитые районы снижают или полностью обрывают производство.

Хроника длится 12 циклов. Для победы нужно сохранить целостность и сцепление города, завершить минимум три великих проекта и пережить финальную Белую бурю.

## Вариативность

- три стартовые хартии меняют экономику и правило поворота;
- код города детерминирует процедурную раскладку колец, фронты и события;
- пять великих проектов создают разные экономические и оборонительные сборки;
- пять разовых приказов позволяют ремонтировать районы, укреплять фронт, менять прогноз и форсировать привод;
- шесть событий предлагают выбор с немедленными и отложенными последствиями.

Карта управляется свайпом непосредственно по нужному кольцу. Для доступности те же действия продублированы выбором кольца и кнопками поворота; на клавиатуре работают `1`–`3` и стрелки.

## Визуальная система

СВОД использует язык инженерного атласа: тёплая бумага, кобальтовая тушь, керамические сектора и сигнальный оранжевый. Кольца, входы, выходы, поток, повреждения и прогноз бури отрисовываются собственным high-DPI Canvas-рендерером без внешних ассетов. Звук процедурный и отключаемый; вибрация и уменьшенная анимация настраиваются отдельно.

## Идентичность

- Slug: `svod`
- Version: `0.1.0`
- Cache: `svod-v0.1.0`
- Storage: `pocket-works:svod`
- Orientation: `portrait`

## Границы приложения

Все исходники, визуал, звук, сохранения, PWA-манифест и offline shell принадлежат `apps/svod/`. Приложение использует общий мобильный runtime Pocket Works, но не меняет платформенные файлы.

Чистая симуляция находится в `game-core.js`; DOM, Canvas, жесты, звук и сохранение — в `app.js`. Это позволяет тестировать экономику, повороты, события и финал без браузера.

## Проверка

Из корня репозитория:

```bash
node apps/svod/test.mjs
npm run registry:check
npm run prepare:site
npm run validate:site
```

Браузерный smoke-сценарий проверяет размер iPhone, бесплатный поворот Цехового пакта, свайп по кольцу, проекты, завершение цикла, сохранение после перезагрузки, финальную запись и offline reload через Service Worker.
